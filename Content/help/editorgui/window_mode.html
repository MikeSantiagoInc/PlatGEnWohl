Sub-Windows/Tabs view mode

You can switch view to Tab view, and all opened documents will be as Tabs:

 

 

You have two ways for switch between sub-windows and tabs view mode:

  • In Window menu you can select item "Sub Windows" or "Tab Windows" for switch to necessary view mode

  • In application settings in "Main" tab tou can check switch in "Document view" group.
